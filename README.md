**MauiAppHotel**

Aplicação desenvolvida em .NET MAUI para fins educacionais, simulando um sistema de hotel com tela principal e uma tela Sobre contendo informações do desenvolvedor.

**Funcionalidades**

Interface construída com XAML
Navegação entre telas
Tela Sobre exibindo:
Nome do desenvolvedor
Ano de desenvolvimento


| Informação                 | Valor      |
| -------------------------- | ---------- |
| **Nome**                   | Luan Costa |
| **Ano de desenvolvimento** | 2025       |

**Objetivo do Projeto**
Atender à atividade proposta pela disciplina, que contempla:

Desenvolvimento do layout solicitado
Personalização do projeto
Criação da tela Sobre
Inserção das informações do desenvolvedor
